最大公因数与线性方程

$$
gcd(a, b) = gcd(b, a\%b) \
ax + by = bx + a\%b y \
= bx + (a - (a/b)b)y \
= bx + ay-(a/b)by\
= ay + b(x-(a/b)y) \
$$

推导出，每次的 $x = y$, $y = (x-(a/b)y)$
